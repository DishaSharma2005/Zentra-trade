# 🚀 Zentra Trade  
### AI-Powered Portfolio Management System (Inspired by Zerodha)

Zentra Trade is a full-stack stock trading platform that simulates real-world trading workflows.  
It enables users to track live stock prices, manage portfolios, and execute trades through a clean and responsive interface.

## ✨ Features

- 🔐 Secure authentication (including Google login)  
- 📈 Live stock price tracking  
- 💰 Buy & Sell functionality with order cancellation window  
- 📊 Portfolio management (holdings, transactions, funds)  
- 📉 Dashboard with profit/loss tracking and wallet balance  
- 🤖 AI-powered chatbot (Gemini API integration)  
- 💳 Add funds using Stripe (test mode)  
- 📱 Fully responsive (mobile + desktop)

---

## 🛠️ Tech Stack

**Frontend:**  
React.js  

**Backend:**  
Node.js, Express.js  

**Database:**  
Supabase  

**APIs & Integrations:**  
Gemini API (AI chatbot)  
Stripe API (payments)  
REST APIs

## ⚙️ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/DishaSharma2005/Zentra-Trade.git
cd Zentra-Trade

2. Install dependencies
Bash
npm install
3. Setup environment variables
Create a .env file and add:

REACT_APP_API_URL=your_backend_url
GEMINI_API_KEY=your_api_key
STRIPE_KEY=your_stripe_key
4. Run the project
Bash
npm start
```
